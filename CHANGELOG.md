# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2025-12-14

### Added

- Initial release of MMM-Fuel-NSW MagicMirror module
- Real-time fuel price display from NSW Fuel Check API
- Support for multiple fuel types (P95, E10, etc.)
- Brand filtering and sorting by price/distance
- Configurable display options (logos, addresses, status)
- TypeScript source code with build system
- Comprehensive test suite
- ESLint and Prettier code formatting
